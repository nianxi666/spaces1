支付设置 (Payhip)

基本配置

配置 Payhip 支付链接和验证密钥。

开启后，原有的“人工审核”页面将被替换为 Payhip 支付引导。

请填写 Payhip 商品页面的完整链接。

请在 Payhip 后台的 Webhook 设置中，将此密钥作为参数添加到 URL 末尾。
例如: https://your-domain.com/api/payment/payhip/webhook?key=YOUR_SECRET_KEY

填写后，如果 Webhook 未触发（例如使用了 100% 优惠码），系统将主动查询 Payhip API 来确认订单。
您可以在 Payhip 后台的 "Settings → API" 中获取 API Key。
