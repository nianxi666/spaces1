## 🚨 常见错误

### 错误 1："用户不存在"

**原因**：用户名填写错误或用户未注册

**解决**：
- 检查用户名拼写
- 确认用户已注册

### 错误 2："订单已处理"

**原因**：该订单已经被激活过

**解决**：
- 这是正常的，说明订单已处理
- 无需重复激活

### 错误 3："格式错误"

**原因**：CSV 格式不正确

**解决**：
- 确保使用英文逗号分隔
- 每行一个订单
- 不要有多余的空格

## 💡 提示

### 格式说明

```
用户名,订单ID,邮箱
```

- **用户名**：必填
- **订单ID**：必填（从 Payhip 复制）
- **邮箱**：可选（建议填写）

### 示例

正确的格式：
```
alice,LWRwLk8Pza,alice@example.com
bob,KzyO74ekGX
charlie,2GZxenZ6zp,charlie@example.com
```

错误的格式：
```
alice LWRwLk8Pza  // ❌ 没有逗号
alice, LWRwLk8Pza  // ❌ 逗号后有空格
,LWRwLk8Pza,alice@example.com  // ❌ 缺少用户名
```
